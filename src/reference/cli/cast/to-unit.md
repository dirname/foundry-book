# cast to-unit

将 ETH 金额转换为其他单位（ether、gwei 或 wei）。

```bash
$ cast to-unit --help
Usage: cast to-unit [VALUE] [UNIT]

Arguments:
  [VALUE]
          要转换的值

  [UNIT]
          要转换到的单位（ether、gwei、wei）
          
          [default: wei]

Options:
  -h, --help
          打印帮助信息（使用 '-h' 查看摘要）
```
