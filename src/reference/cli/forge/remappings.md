# forge remappings

获取项目自动推断的重映射

```bash
$ forge remappings --help
Usage: forge remappings [OPTIONS]

Options:
      --root <PATH>
          项目的根路径。
          
          默认情况下是 Git 仓库的根目录（如果在 Git 仓库中），或者当前工作目录。

      --pretty
          以分组形式美观地打印重映射

  -h, --help
          打印帮助信息（使用 '-h' 查看摘要）
```
