## cast wallet verify

### NAME

cast-wallet-verify - 验证消息的签名。

### SYNOPSIS

``cast wallet verify`` [*options*] `--address` *address* *message* *signature*

### DESCRIPTION

验证消息的签名。

### OPTIONS

#### 签名选项

`-a` *address*  
`--address` *address*
