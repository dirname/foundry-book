## cast from-wei

### NAME

cast-from-wei - 将 wei 转换为 ETH 金额。

### SYNOPSIS

``cast from-wei`` [*options*] *value* [*unit*]

### DESCRIPTION

将 wei 转换为 ETH 金额。
