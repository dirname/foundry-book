## cast to-wei

### NAME

cast-to-wei - 将以太币数量转换为wei。

### SYNOPSIS

``cast to-wei`` [*options*] *value* [*unit*]

### DESCRIPTION

将以太币数量转换为wei。

建议使用 [`cast to-unit`](./cast-to-unit.md)。
