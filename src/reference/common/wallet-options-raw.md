```markdown
#### 钱包选项 - 原始:
`-i`  
`--interactive <NUM>`  
&nbsp;&nbsp;&nbsp;&nbsp; 打开一个交互式提示符以输入您的私钥。接受一个值，表示要输入的私钥数量。  
&nbsp;&nbsp;&nbsp;&nbsp; 默认为 `0`。  

`--mnemonic-derivation-path <PATHS>`  
&nbsp;&nbsp;&nbsp;&nbsp; 钱包派生路径。适用于 `--mnemonic-path` 和硬件钱包。  

`--mnemonic-indexes <INDEXES>`  
&nbsp;&nbsp;&nbsp;&nbsp; 使用给定助记词索引的私钥。与 `--mnemonic-paths` 一起使用。  
&nbsp;&nbsp;&nbsp;&nbsp; 默认为 `0`。

`--mnemonic-passphrase <PASSPHRASE>`  
&nbsp;&nbsp;&nbsp;&nbsp; 使用 BIP39 密码短语。  

`--mnemonic <PATHS>`  
&nbsp;&nbsp;&nbsp;&nbsp; 使用指定路径的助记词短语或助记词文件。  

`--private-key <RAW_PRIVATE_KEY>`  
&nbsp;&nbsp;&nbsp;&nbsp; 使用提供的私钥。

`--private-keys <RAW_PRIVATE_KEYS>`  
&nbsp;&nbsp;&nbsp;&nbsp; 使用提供的私钥。
```
